# Suporte Brasil Locadora - Website

Este é o site estático da Suporte Brasil Locadora, desenvolvido com HTML, CSS e JavaScript.

## Estrutura do Projeto

- `index.html`: Página principal.
- `style.css`: Estilos globais e responsivos.
- `main.js`: Lógica de interação (menu mobile, filtros, scroll).
- `assets/img/`: Imagens dos veículos e assets.

## Como Publicar na Vercel

Este projeto está pronto para publicação na Vercel sem configurações complexas.

1. Faça o commit de todas as alterações para o seu repositório GitHub.
2. Acesse [vercel.com](https://vercel.com) e faça login.
3. Clique em "Add New Project" e selecione "Mport Git Repository".
4. Escolha este repositório.
5. A Vercel detectará automaticamente que é um projeto estático.
6. Clique em "Deploy".

## Tecnologias

- HTML5 HTML Semântico
- CSS3 (Variáveis, Flexbox, Grid, Animações)
- JavaScript (ES6+)
- Fonte: Outfit (Google Fonts)
- Ícones: Font Awesome
